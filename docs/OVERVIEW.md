# StillHungry - 프로젝트 개요

## 서비스 소개

구로디지털단지 및 가산디지털단지 내 구내식당들의 메뉴를 한눈에 확인할 수 있는 웹 서비스.

직장인들이 오늘 어느 식당에서 무엇을 먹을지 빠르게 확인하고, 식당 관리자는 손쉽게 메뉴를 등록·관리할 수 있다.

---

## 대상 지역

- 구로디지털단지 (서울 구로구)
- 가산디지털단지 (서울 금천구)

---

## 사용자 구분

| 구분 | 역할 |
|------|------|
| Client (일반 사용자) | 식당 메뉴 조회, 공지/이벤트 확인 |
| Admin (식당 관리자) | 자기 식당의 메뉴·공지·이벤트 등록 및 관리 |
| Super Admin | 전체 식당 관리, Admin 계정 발급, 모든 데이터 CRUD |

---

## 주요 기능

### Client
- 단지별 식당 목록 조회
- 식당 선택 후 날짜별 메뉴 조회 (조식 / 중식 / 석식)
- 이벤트 및 공지사항 확인

### Admin
- 날짜 및 식사 유형별 메뉴 등록·수정·삭제
- 공지사항·이벤트 등록·수정·삭제
- 자기 식당 데이터만 접근 가능

### Super Admin
- 단지·식당 생성 및 관리
- Admin 계정 발급 및 권한 관리
- 전체 식당의 모든 데이터 CRUD

---
